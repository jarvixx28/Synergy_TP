# Comparison Report

## Objective

This report compares the summaries generated by the manual CSV parser and the pandas-based parser.

## Comparison

Both implementations produced equivalent results for:

- Total number of students
- Number of submitted students
- Number of missing submissions
- Average score
- Highest scorer
- Lowest scorer among submitted students
- Domain-wise average score
- Students who did not submit
- Students scoring below 5

The outputs match successfully. Any difference in dictionary key order does not affect the correctness of the generated summaries.

## Conclusion

The manual parser correctly reproduces the same analysis performed using pandas for the given dataset.
