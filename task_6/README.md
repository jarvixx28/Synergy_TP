# Task 6 – Technical Report

## Objective

The objective of Task 6 was to prepare a technical report explaining the concepts learned throughout Tasks 1–5 of the Software & Machine Learning Taskphase.

The report focuses on the theoretical concepts, design decisions, and practical observations behind the implementation rather than explaining the source code line by line.

* Abstract
* Introduction
* Development Environment & Version Control

* CSV Parsing and Pandas
* Data Cleaning and Validation
* Data Visualization using Matplotlib
* Reflection
* Conclusion
* References

The report also contains:

* Summary table of Tasks 1–5
* Workflow diagram
* Data cleaning rules table
