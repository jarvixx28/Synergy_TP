# Linux Commands Documentation

## 1. pwd

Command:

```bash
pwd
```

Purpose:
Displays current working directory.

Example Output:

```text
/home/user/Synergy_TP
```

---

## 2. ls

Command:

```bash
ls
```

Purpose:
Lists files and folders.

Example Output:

```text
task_1 task_2
```

---

## 3. ls -la

Command:

```bash
ls -la
```

Purpose:
Lists all files including hidden files.

---

## 4. cd

Command:

```bash
cd task_1
```

Purpose:
Changes directory.

---

## 5. mkdir

Command:

```bash
mkdir demo
```

Purpose:
Creates a new directory.

---

## 6. touch

Command:

```bash
touch file.txt
```

Purpose:
Creates an empty file.

---

## 7. cat

Command:

```bash
cat file.txt
```

Purpose:
Displays file contents.

---

## 8. echo

Command:

```bash
echo Hello
```

Purpose:
Prints text to terminal.

Output:

```text
Hello
```

---

## 9. cp

Command:

```bash
cp file.txt copy.txt
```

Purpose:
Copies files.

---

## 10. mv

Command:

```bash
mv copy.txt newcopy.txt
```

Purpose:
Moves or renames files.

---

## 11. rm

Command:

```bash
rm file.txt
```

Purpose:
Deletes files.

---

## 12. grep

Command:

```bash
grep Hello sample.txt
```

Purpose:
Searches text in files.

---

## 13. find

Command:

```bash
find . -name "*.py"
```

Purpose:
Searches for files.

---

## 14. head

Command:

```bash
head sample.txt
```

Purpose:
Displays first lines of a file.

---

## 15. tail

Command:

```bash
tail sample.txt
```

Purpose:
Displays last lines of a file.

---

## 16. wc

Command:

```bash
wc sample.txt
```

Purpose:
Counts lines, words and characters.

---

## 17. chmod

Command:

```bash
chmod +x script.sh
```

Purpose:
Changes file permissions.
